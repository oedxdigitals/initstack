# Initstack v2.0.0 Roadmap

## Goals
- Make Initstack fully extensible
- Improve developer experience
- Enable CI / non-interactive usage

## Planned Features
- Interactive template variables
- Remote template repositories
- Plugin install / update commands
- Template inheritance
- Non-interactive mode (--yes)
- Initstack config file
- Doctor auto-fix mode
- Improved logging system

## Stability
- Maintain backward compatibility where possible
- Clear migration guide from v1.x to v2.x
